Introducción

Sistema completo para gestión de matrículas estudiantiles, cursos y relaciones académicas. Desarrollado en C++ con documentación automática mediante Doxygen.

Características Principales

  • Registro de estudiantes y cursos
  • Proceso de matrícula automatizado
  • Generación de reportes
  • Sistema modular y escalable

Cómo Usar

  1. Clona el repositorio
  2. Compila con g++ -Iinclude src/*.cpp -o sistema_matricula
  3. Ejecuta ./sistema_matricula

Estructura del Proyecto

  • src/: Código fuente
  • include/: Headers
  • docs/: Documentación generada
